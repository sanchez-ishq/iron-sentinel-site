## Official lab / hardware & software list

CompTIA's sample lab list — useful for hands-on practice (the exam is partly performance-based).

**Equipment:** Laptops · Cloud VMs · GPUs (e.g., NVIDIA Jetson Orin Nano) · Mobile devices · Sandbox environment · LAN

**Software:**
- Virtual containers
- Large data sets / Test data sets
- **Python** environment · **R** environment · **Jupyter** environment · IDE
- Chatbots · **LLMs**
- Open-source tools: **GitHub**, **Ollama**
- Cloud-based environment / Cloud-based AI studios
- **Vector database** · **NoSQL database** · **Neo4j** graph database

> **Practical tip:** if you want hands-on reps, a local **Ollama** + a small open model, a **vector DB** (e.g., Chroma/pgvector), and a **Jupyter** notebook reproduce most of what the performance-based items assume — you can practice prompt injection, RAG access control, and guardrail testing locally in a sandbox.
